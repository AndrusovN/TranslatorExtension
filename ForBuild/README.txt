Для chrome:
заходите по ссылке chrome://extensions/
нажимаете "загрузить распакованное расширение"
и выбираете всю эту папку

Для Mozilla:
заходите по ссылке about:debugging#/runtime/this-firefox
Нажимаете "загрузить временное расширение"
и выбираете файл "manifest.json"
